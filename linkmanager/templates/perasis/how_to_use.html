Firefox:

History: (skip to how-to if you want it done)

Since Firefox 41 it's no longer possible to customize the page shown in a new tab via changing the preference browser.newtab.url in about.config. Because of the fact that hijackers abused the preference in the past, Mozilla decided to remove it. Fortunately, by removing it, Mozilla also introduced a new API to bring this functionality back to life as an add-on.


How-to:

Click this link (this is one of many Firefox addons that allows you to place amibit.org on new tab, choosen randomly by us).

https://addons.mozilla.org/en-US/firefox/addon/new-tab-override/?src=search

Add to firefox
On top left corner Firefox will ask you do you want to install. Click add.

On top right click the icon and you will arrive to Setting page.

Option should be "custom URL" already set so don't chane that.

For URL type : "www.amibit.org"

Focus - We recommend you set focus Amibit so check the box, but its entirely up to you.
